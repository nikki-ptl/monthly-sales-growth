# 📊 Monthly Sales Growth Dashboard

This project is a Power BI dashboard that analyzes sales and profit trends.

## 🔹 Features

* Monthly profit analysis
* Sales distribution by product
* Regional profit comparison
* KPI indicators (Total Sales, Profit, % Profit)

## 🛠 Tools Used

* Power BI
* MySQL / Excel dataset

## 📁 Files

* `MonthlySales.pbix` → Power BI dashboard
* `sales_data.csv` → dataset
